Changed Jekyll theme to end2end! - Sun, Jan 24, 2021

Last week, I suddenly felt the urge to change my personal web hosted in github. I had it running for quite sometime with HTML5 UP but it was time for some change!!

Then I decided to try out Jekyll themes as Github supports it. Turns out there is a huge collection of themes on their site. Check out the Jekyll docs for more info on how to get the most out of Jekyll.

Couple of themes caught my eyes. end2end is what I decided to try. It’s been so easy to set up and get started with. Kudos to [Fernando Moreira] Fernando-Moreira, the creator of the theme.
